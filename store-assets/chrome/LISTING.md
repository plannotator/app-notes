# Chrome Web Store listing

## Short description

Annotate websites, attach notes to page elements, and export all notes for a site at once.

## Full description

App Notes helps you review websites without losing track of what needs attention.

Click any element on a web page and add a note. Your notes stay saved as you move between pages. When you are finished, view, copy, or export every note for the site in one place.

Use App Notes for website reviews, quality assurance, design and development feedback, bug tracking, research, content review, and working with coding agents.

Everything stays in your browser. App Notes has no account, analytics, advertising, or cloud storage.
